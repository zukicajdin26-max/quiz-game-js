# 🎮 Ultimate Quiz Game

Ultimate Quiz Game is an interactive browser-based quiz built with HTML, CSS, and JavaScript. It challenges players with randomized questions, a countdown timer, and a scoring system to make gameplay exciting and competitive.

---

## 🚀 Live Demo

👉 https://zukicajdin26-max.github.io/quiz-game-js/

---

## 📌 Features

* ⏱ Timed questions for extra challenge
* 🧠 Randomized quiz questions
* 📊 Score tracking system
* 🔊 Sound effects for better experience
* 🏆 Local leaderboard (stored in browser)

---

## 🛠️ Built With

* HTML
* CSS
* JavaScript

1. Clone the repository:

```bash

```

2. Open the project folder

3. Run the game by opening:

```bash
index.html
```
